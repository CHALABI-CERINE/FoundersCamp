# 🚀 FoundersCamp

A modern, interactive website for FoundersCamp - an entrepreneurship and startup community platform.

## ✨ Overview

FoundersCamp is a web application designed to connect founders, entrepreneurs, and startup enthusiasts. The platform provides resources, networking opportunities, and tools for building successful startups.

## 🎯 Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Interactive UI**: Engaging user interface with smooth animations
- **Modern Styling**: Clean and professional design using custom CSS
- **Dynamic Content**: JavaScript-powered interactive elements

## 🏗️ Project Structure

```
FoundersCamp/
├── index.html         # Main HTML structure
├── style.css          # Custom styling
├── script.js          # Interactive functionality
└── img/               # Image assets
```

## 🛠️ Technology Stack

- **HTML5** (39.4%)
- **CSS3** (36.1%)
- **JavaScript** (24.5%)
- Vanilla JavaScript (no frameworks)

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/CHALABI-CERINE/FoundersCamp.git
   cd FoundersCamp
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```
